# FastAPI Tasks API

## Создание виртуального окружения

```
python -m venv .venv
```

## Установка зависимостей

```
python -m pip install -r requirements.txt
```

## Запуск приложения

```
python -m uvicorn app.main:app --reload
```

## Запуск тестов

```
python -m pytest
```

## Запуск Docker

```
docker compose up --build
```
